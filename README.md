Ribosome360

Interactive DNA Transcription, Translation and Mutation Simulator for Biology and Bioinformatics Education.

Features

- DNA to mRNA transcription simulation
- mRNA to protein translation visualization
- Mutation analysis sandbox
- Interactive codon wheel
- Amino acid statistics dashboard
- Biology quiz module

Technologies Used

- HTML5
- CSS3
- JavaScript
- SVG Animations
- Chart.js

Educational Purpose

This project helps students understand the Central Dogma of Molecular Biology through interactive simulations and visualizations.
